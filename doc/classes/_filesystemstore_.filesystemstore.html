module

constructor

classdesc

Store for MongoPortable (https://github.com/EastolfiWebDev/MongoPortable)

param

Additional options

param

The name of the directory where the database will be located

param

Set it false to make all the file access asynchronous. (Currently only sync=true is supported)

param

The extension of the collection files. (Currently only "json" is supported)
